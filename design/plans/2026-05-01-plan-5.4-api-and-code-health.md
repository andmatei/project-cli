- [ ] **Step 1: Implement**

In `src/keel/workspace.py`:
```python
def iter_projects() -> Iterator[tuple[str, ProjectManifest, str]]:
    """Yield (project_name, manifest, current_phase) for each project in PROJECTS_DIR.

    Skips entries that don't have a `design/project.toml`. Useful for cross-project
    tooling (status dashboards, exports, plugins).
    """
    pdir = projects_dir()
    if not pdir.is_dir():
        return
    for entry in sorted(pdir.iterdir()):
        if not entry.is_dir():
            continue
        manifest_path = entry / "design" / "project.toml"
        if not manifest_path.is_file():
            continue
        try:
            from keel.manifest import load_project_manifest
            manifest = load_project_manifest(manifest_path)
        except Exception:
            continue
        phase = read_phase(entry / "design")
        yield (entry.name, manifest, phase)
```

Add `from collections.abc import Iterator` if not present. Keep import of `load_project_manifest` lazy to avoid circular concerns.

## Section 4: Type-narrow find_X / get_X

### Task 4.1: Split into `find_X` (Optional) + `get_X` (raises)

**Files:**
- Modify: `src/keel/manifest/queries.py` (add `get_milestone` / `get_task`)
- Modify: `src/keel/api.py` (export the new functions)
- Modify: command files that currently do `find_X` + `out.fail` (sweep to `get_X`)

The current `find_milestone`/`find_task` return `Optional[T]`, but every caller does `if X is None: out.fail(...)`. Type checkers can't narrow through `out.fail` without an `assert`. Splitting gives callers a clean choice: `find_X` for "maybe present" lookups, `get_X` for "must exist" with a clean exit-on-miss.

- [ ] **Step 1: Add `get_milestone` and `get_task`**

In `src/keel/manifest/queries.py`:
```python
def get_milestone(manifest: MilestonesManifest, id: str, *, out: Output) -> Milestone:
    """Return the milestone with the given id, or call out.fail() with NOT_FOUND."""
    m = find_milestone(manifest, id)
    if m is None:
        from keel.errors import ErrorCode
        out.fail(f"no milestone with id '{id}'", code=ErrorCode.NOT_FOUND)
    return m


def get_task(manifest: MilestonesManifest, id: str, *, out: Output) -> Task:
    """Return the task with the given id, or call out.fail() with NOT_FOUND."""
    t = find_task(manifest, id)
    if t is None:
        from keel.errors import ErrorCode
        out.fail(f"no task with id '{id}'", code=ErrorCode.NOT_FOUND)
    return t
```

`Output` import goes via TYPE_CHECKING to avoid a hard cycle. Type the parameter as `"Output"`.

- [ ] **Step 2: Add to `keel.api`**

```python
from keel.manifest import get_milestone, get_task
```
Add to `__all__` under Manifest.

- [ ] **Step 3: Sweep callers**

